# Statistics Basics for Data Science

This repo contains summary of the things I explored to learn statistics from scratch

## Basics Covered(`statistics_basics.ipynb`):

- Using libraries like Seaborn, numpy, matplotlib and statistics
- Computing Mean, Median, Mode
- Five Number Summary computation
  - Minimum
  - First Quartile (Q1)
  - Median
  - Third Quartile (Q3)
  - Maximum
- Identifying outliers using simple BoxPlot
- Identifying the type of Distribution
- Examples of Normal Distributions
- Plotting Histograms and Bar Plots

## Methods of identifying Outliers(`Outliers.ipynb`):

- By basic plots
- Z-score Computation
- Finding IQR
- Box Plot

## Hypothesis Testing(`Hypothesis_Testing.ipynb`):

- Chi-Square Test
- Null and alternate Hypothesis
- T-Test - One and two sample testing
- Paired T-Test
- Correlation
- Anova Test (F-test)

## Project Structure

- `excercise.ipynb` - Practice problem for hypothesis testing
- `Hypothesis_Testing.ipynb` - Statistical hypothesis testing examples
- `Outliers.ipynb` - Outlier detection and analysis
- `statistics_basics.ipynb` - Fundamental statistics concepts
- `Readme.md` - Project documentation
- `.gitignore` - Git ignore rules
